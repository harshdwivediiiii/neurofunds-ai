## ğŸ Getting Started

First, clone the project:

```bash
git clone https://github.com/harshdwivediiiii/neurofunds-ai.git
cd neurofunds-ai
```

Install dependencies:

```bash
pnpm install
# or
npm install
# or
yarn install
```

Run the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

Visit: [http://localhost:3000](http://localhost:3000)
